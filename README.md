# Sky Climber

Платформер с тремя уровнями, дэшем и таймером. Доберись до звезды на каждом уровне как можно быстрее.

## Запуск

**Готовый .exe (рекомендуется):**
Скачай `SkyClimber.exe` из [Releases](../../releases) — Python устанавливать не нужно.

## Управление

| Действие | Клавиши |
|----------|---------|
| Движение | `←` `→` или `A` `D` |
| Прыжок | `Z`, `Пробел`, `↑` или `W` |
| Дэш | `X` или `Shift` |
| Рестарт | `R` |
| Выход | `Escape` |

**Прыжок с удержанием** — чем дольше держишь кнопку, тем выше прыгаешь.

**Дэш** — рывок в направлении движения. Перезаряжается при касании земли. Можно направлять в любую из 8 сторон.

**Wall jump** — прыжок от стены: прижмись к стене в воздухе и нажми прыжок.

## Уровни

Три уровня с нарастающей сложностью. Цель на каждом — добраться до мерцающей золотой звезды.

- **Уровень 1** — знакомство с механиками, прямой маршрут вверх
- **Уровень 2** — зигзагообразный путь, нужен wall jump
- **Уровень 3** — длинная цепочка мелких платформ, важен дэш

## Таймер и рекорды

- Общий таймер идёт с первого уровня до финиша
- Лучшее время (PB) сохраняется до закрытия игры
- На экране результатов видны сплиты по каждому уровню и отклонение от рекорда
